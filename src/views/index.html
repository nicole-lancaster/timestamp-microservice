Nicole's Timestamp Microservice

This is a microservice that allows you to input a date as an endpoint. You will then receive a timestamp in JSON format!

Example Usage:

  • https://replit.com/@nicole-lancaste/timestamp-microservice/api/2015-12-25
  • https://replit.com/@nicole-lancaste/timestamp-microservice/api/1451001600000

Example Output:

{"unix":1451001600000, "utc":"Fri, 25 Dec 2015 00:00:00 GMT"}
